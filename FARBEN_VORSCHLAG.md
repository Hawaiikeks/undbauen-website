# Farbvorschlag für UX - Innovative & Hippe Farbpalette

## Option 1: Warm & Energetisch (Empfohlen für Innovationsnetzwerk)
```css
:root {
  /* Light Mode */
  --primary: #FF6B35;        /* Warmes Orange - Energie & Innovation */
  --primary-hover: #E55A2B;
  --secondary: #4ECDC4;     /* Türkis - Modern & Fresh */
  --accent: #FFE66D;         /* Gelb - Optimismus & Kreativität */
  
  --bg: #FFF8F3;             /* Warmes Weiß */
  --surface: #FFFFFF;
  --border: #FFE5D9;         /* Sanftes Orange */
  
  --text-primary: #2C3E50;   /* Dunkles Blau-Grau */
  --text-secondary: #5D6D7E;
  --text-muted: #A0A0A0;
}

[data-theme="dark"] {
  --primary: #FF8C5A;        /* Helleres Orange für Dark Mode */
  --primary-hover: #FF6B35;
  --secondary: #6EDDD6;
  --accent: #FFF89C;
  
  --bg: #1A1A1A;
  --surface: #2D2D2D;
  --border: #4A4A4A;
  
  --text-primary: #F5F5F5;
  --text-secondary: #D0D0D0;
  --text-muted: #888888;
}
```

## Empfehlung
**Option 1 (Warm & Energetisch)** passt am besten zu einem Innovationsnetzwerk:
- Orange vermittelt Energie, Innovation und Wärme
- Türkis bringt Modernität und Frische
- Gelb-Akzente für Optimismus
- Warme Hintergründe schaffen eine einladende Atmosphäre

Die Farben sind:
- ✅ Modern und innovativ
- ✅ Einladend und warm
- ✅ Professionell aber nicht steif
- ✅ Gute Lesbarkeit (WCAG-konform)
- ✅ Funktioniert in Light & Dark Mode
